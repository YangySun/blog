# 欢迎来到我的博客

这是我的第一篇博客文章，欢迎大家访问！

## 关于这个博客

这个博客使用以下技术栈构建：

- **Vue.js 3**：使用Composition API
- **Vite**：快速的开发服务器和构建工具
- **Markdown**：文章内容使用Markdown格式编写
- **GitHub Pages**：托管在GitHub上

## 功能特点

- 文章列表和详情展示
- 个人信息展示
- 友情链接
- 响应式设计
- 主题切换（亮色/暗色）
- 文章搜索功能
