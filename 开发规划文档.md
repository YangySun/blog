## 2. 技术选型

### 2.1 核心框架
- **Vue 3**：使用Composition API，更好的TypeScript支持
- **Vite**：快速的开发服务器和构建工具
- **Vue Router**：路由管理
- **Pinia**：状态管理（替代Vuex）

### 2.2 Markdown处理
- **marked**：Markdown解析
- **highlight.js**：代码高亮
- **markdown-it-anchor**：生成锚点
- **markdown-it-toc-done-right**：生成目录

### 2.3 UI样式
- **SCSS**：CSS预处理器
- **CSS Variables**：主题切换支持

### 2.4 其他工具
- **dayjs**：日期处理
- **lunr.js**：全文搜索

## 4. 关键技术实现

### 4.1 Markdown文章加载
```javascript
// 使用Vite的import.meta.glob动态加载Markdown文件
const articles = Object.entries(
  import.meta.glob('/src/data/articles/*.md', { as: 'raw' })
).map(([path, content]) => {
  const { frontmatter, body } = parseMarkdown(content)
  return {
    id: path.match(/\/([^/]+)\.md$/)[1],
    ...frontmatter,
    content: body
  }
})
```

### 4.2 主题切换
```javascript
// 使用CSS Variables实现主题切换
:root {
  --bg-primary: #ffffff;
  --text-primary: #333333;
  --accent-color: #2196F3;
}

[data-theme='dark'] {
  --bg-primary: #1a1a1a;
  --text-primary: #ffffff;
}
```

### 4.3 全文搜索
```javascript
// 使用lunr.js实现全文搜索
const index = lunr(function() {
  this.field('title', { boost: 10 })
  this.field('content')
  articles.forEach(article => {
    this.add(article)
  })
})
```

## 6. Git工作流

### 6.1 分支策略
- `main`：生产环境分支
- `develop`：开发分支
- `feature/*`：功能分支
- `bugfix/*`：修复分支

### 6.2 提交规范
```
feat: 新功能
fix: 修复bug
docs: 文档更新
style: 代码格式调整
refactor: 重构
perf: 性能优化
test: 测试相关
chore: 构建/工具相关
```

## 7. 部署流程
